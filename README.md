# 💼 Personal Portfolio Website

A modern and responsive personal portfolio website showcasing my projects, technical skills, education, certifications, internship experience, and achievements. The website is designed to present my professional profile and software development journey in an interactive and visually appealing way.

## ✨ Features

- 👨‍💻 About Me
- 📄 Resume Download
- 🚀 Featured Projects
- 💻 Technical Skills
- 🏆 Certifications & Achievements
- 📚 Education Details
- 📞 Contact Section
- 🌐 Social Media Links
- 📱 Fully Responsive Design
- 🎨 Modern UI/UX

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Bootstrap
- Font Awesome
- GitHub Pages (Deployment)

## 🎯 Purpose

This portfolio was developed to showcase my technical skills, projects, and professional journey as a B.Tech Information Technology student. It serves as a central hub for recruiters, collaborators, and visitors to explore my work and connect with me.
